## About

GradCapFinder's Site is a landing page for GradCapFinder (An Educational Consulting LLC focused on supporting master's and doctoral level students finish their degrees).

## Tech Stack

-   React
-   Typescript
-   Gatsby.js
-   Chakra UI
-   Netlify
